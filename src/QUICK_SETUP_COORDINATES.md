# ⭐ วิธีแก้พิกัด - Quick Setup Guide

---

## 🎯 ไฟล์ที่ต้องแก้: `weatherService.js`

### 📍 ตำแหน่งที่ต้องแก้ (บรรทัด 11-13)

```javascript
// ====================================================================
// 🔧 CONFIGURATION - แก้พิกัดตรงนี้
// ====================================================================
const WEATHER_CONFIG = {
  // พิกัดโรงงาน: 13°28'26.1"N 101°05'22.3"E
  LATITUDE: 13.473917,   // ⭐⭐⭐ แก้พิกัด Latitude ตรงนี้ ⭐⭐⭐
  LONGITUDE: 101.089528, // ⭐⭐⭐ แก้พิกัด Longitude ตรงนี้ ⭐⭐⭐
  
  // TMD API Configuration
  API_BASE_URL: 'https://data.tmd.go.th/nwpapi/v1',
  API_TOKEN: 'eyJ0eXAiOiJKV1QiLCJhbGc...', // ← ไม่ต้องแก้
  
  // Update Interval
  UPDATE_INTERVAL_MS: 30 * 60 * 1000, // ← ไม่ต้องแก้ (30 นาที)
  
  // ... ส่วนอื่นๆ
};
```

## 🔄 แปลงพิกัดจาก DMS → Decimal

### พิกัดปัจจุบัน: `13°28'26.1"N 101°05'22.3"E`

| รูปแบบ | Latitude | Longitude |
|--------|----------|-----------|
| **DMS** | 13°28'26.1"N | 101°05'22.3"E |
| **Decimal** ✅ | **13.473917** | **101.089528** |

### วิธีแปลงด้วยตัวเอง:

**สูตร:**
```
Decimal = Degrees + (Minutes / 60) + (Seconds / 3600)
```

**ตัวอย่าง Latitude:**
```
13°28'26.1"N
= 13 + (28/60) + (26.1/3600)
= 13 + 0.4666667 + 0.0072500
= 13.473917
```

**ตัวอย่าง Longitude:**
```
101°05'22.3"E
= 101 + (5/60) + (22.3/3600)
= 101 + 0.0833333 + 0.0061944
= 101.089528
```

## ✅ Checklist การตั้งค่า

- [ ] **เปิดไฟล์** `weatherService.js`
- [ ] **ค้นหาบรรทัด** `LATITUDE:` และ `LONGITUDE:`
- [ ] **แทนที่ตัวเลข** ด้วยพิกัดของคุณ (รูปแบบ Decimal)
- [ ] **บันทึกไฟล์**
- [ ] **Refresh Browser** และเปิด Console (`F12`)
- [ ] **ตรวจสอบ Log** ว่ามีข้อความ:
  ```
  📍 Location: 13.473917°N, 101.089528°E
  ✅ Weather data updated successfully
  ```

## 🔍 วิธีหาพิกัดจาก Google Maps

1. เปิด **Google Maps**: https://maps.google.com
2. **คลิกขวา** ที่ตำแหน่งที่ต้องการ
3. เลือก **"What's here?"** หรือ **"มีอะไรที่นี่"**
4. **คัดลอกพิกัด** ที่แสดงด้านล่าง (รูปแบบ: `13.473917, 101.089528`)
5. **นำมาใส่** ในไฟล์ `weatherService.js`

**ตัวอย่างหน้าจอ Google Maps:**
```
┌─────────────────────────┐
│  📍 13.473917, 101.089528 │ ← คัดลอกตรงนี้
└─────────────────────────┘
```

## 🚨 ข้อควรระวัง

### ❌ ผิด (DMS Format):
```javascript
LATITUDE: "13°28'26.1\"N",  // ผิด! ห้ามใช้สตริง
LONGITUDE: "101°05'22.3\"E", // ผิด!
```

### ✅ ถูกต้อง (Decimal Format):
```javascript
LATITUDE: 13.473917,   // ถูกต้อง!
LONGITUDE: 101.089528, // ถูกต้อง!
```

### หมายเหตุ:
- ใช้เครื่องหมาย `.` (จุด) ไม่ใช่ `,` (คอมมา)
- ไม่ต้องใส่เครื่องหมายคำพูด `""`
- ไม่ต้องใส่หน่วย `°` หรือ `N`, `E`

## 🎉 เสร็จแล้ว!

หลังจากแก้พิกัดแล้ว ระบบจะดึงข้อมูลอากาศจากพิกัดที่คุณระบุ

**ทดสอบ:**
1. Refresh Browser
2. ดูข้อมูลใน Weather Widget
3. กดปุ่ม 🔄 Refresh เพื่อดึงข้อมูลใหม่

---

## 📞 ต้องการความช่วยเหลือ?

- ถ้าพิกัดไม่ถูกต้อง → ตรวจสอบว่าเป็นตัวเลข Decimal
- ถ้าข้อมูลไม่แสดง → เปิด Console (`F12`) ดู Error
- ถ้า API Error → เช็ค Token หมดอายุหรือไม่
